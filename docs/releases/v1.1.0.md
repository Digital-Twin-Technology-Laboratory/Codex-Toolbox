# Codex Toolbox v1.1.0

v1.1.0 将应用内更新与更可靠的本机任务额度估算一起交付。

## 应用内更新

- 集成 Sparkle 2，可选每小时或每天自动检查。
- 新版本在后台下载，准备完成后在设置入口显示红点；点击“立即更新”后安全替换应用并自动重启。
- 更新包同时使用 Sparkle Ed25519 签名和 Apple Developer ID 代码签名校验。
- Release 新增签名 `appcast.xml`，后续正式版本可直接在应用内升级。

> v1.0.0 本身没有 Sparkle，因此升级到 v1.1.0 需手动安装一次。

## 本机任务额度估算

- 根据 rollout 中的逐轮额度快照与 Token 增量建立本机换算率，不再把账户跨设备总消耗直接分摊给本机任务。
- 超过 15 分钟无本机活动后的账户跳变会作为新基线，不归给下一个本机任务。
- 任务估算以 `≈` 与中/低置信度标记；账户实际已用比例仍单独显示并明确“含所有设备”。
- 用量账本只保存时间、Token 增量、窗口时长、已用比例和重置时间，不保存额度 ID、套餐类型或重置卡标识。

## 下载与验证

本 Release 提供已签名、Apple 公证并 staple ticket 的 Universal 2 PKG 与 DMG，同时提供对应 `.sha256` 校验文件。PKG 是从 v1.0.0 手动升级的推荐方式。
